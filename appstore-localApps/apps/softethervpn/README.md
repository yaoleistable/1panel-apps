SoftEther VPN (开发者版主存储库)
- 一个开源的跨平台多协议 VPN 程序
https://www.softether.org/

此存储库包含实验性代码。欢迎提交拉取请求。

稳定版可在以下网址获取：
https://github.com/SoftEtherVPN/SoftEtherVPN_Stable
非开发者用户可以稳定使用。

请注意，一些功能,在稳定版中不可用。

源代码包 (.zip 和 .tar.gz) 以及稳定版的二进制文件也可以在以下网址获取：  
https://www.softether-download.com/
